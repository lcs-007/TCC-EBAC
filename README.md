# TCC - EBAC | Projeto Final QA

Projeto final do curso **Profissão: Engenheiro de Qualidade de Software** da EBAC.

## Objetivo

Este projeto tem como objetivo aplicar na prática os conhecimentos adquiridos ao longo do curso, incluindo:

- Planejamento de testes
- Automação de testes Web
- Automação de testes de API
- Testes de performance
- Integração contínua

## Estrutura do Projeto
TCC-EBAC
│
├── UI → Testes automatizados da interface utilizando Cypress
├── API → Testes automatizados de API
├── performance → Testes de performance
├── jenkins → Integração contínua
└── documentacao → Documentação do projeto

## Ferramentas utilizadas

- Cypress
- Node.js
- Jenkins
- Apache JMeter
- GitHub
